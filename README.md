# PLUMAS AI
Un bot de chat de Telegram AI escrito en Python usando Pyrogram.

A public running instance can be found on telegram as [@plumasaibot](https://t.me/plumasaibot).

<img src="https://github.com/JCystem/jcystem/blob/main/img/GitHub%20Profile.png" width="200" align="right">

## Requirements

- Python 3.7 or higher.
- A [bot token](//t.me/botfather).


## Installation

```sh
$ git clone https://github.com/JCystem/plumas-ai
$ cd plumas
$ pip3 install -U -r requirements.txt
$ cp sample_config.py config.py
```
Edit `config.py` with your own values.
```sh
$ python3 plumas.py
```
